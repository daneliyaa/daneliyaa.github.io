hexo+GitHub博客搭建

搭建过程
第一步：首先安装Node.js和npm：由于Hexo是基于Node.js的静态博客框架，因此需要先安装Node.js和npm。
另外还需安装Git：Hexo使用Git来进行版本控制和部署。
以下是安装前置软件包
Git https://git-scm.com/download/win
VSCode https://code.visualstudio.com/
Node.JS https://nodejs.org/en
Node.js安装完成后，通过以下命令可用来确认安装成功：、
node -v
npm –v
安装Git安装完成后，通过以下命令可用来确认安装成功：
git –version
第二步：安装hexo

  1. 更改npm镜像源
    进入cmd
    npm config set registry https://registry.npmmirror.com
  2. 创建一个文件夹 在里面打开终端
    第三步：使用npm安装hexo
    npm install -g hexo-cli
    第四步：创建博客目录
    hexo init my-blog
    cd my-blog
    npm install
    第五步：启动本地服务器：
    hexo server
    打开浏览器，访问 http://localhost:4000，即可看到默认的Hexo博客页面。
    第六步：配置Hexo
    1.修改配置文件：编辑_config.yml文件，设置站点信息。
    title: illumi激推’ blog

subtitle: illumi激推’ blog
description:
author: daneliya
language: en
timezone: ‘’

搭建过程所遇问题以及解决方法
1．安装Node.js或者npm失败
解决方案：确保是从Node.js官网下载安装包，并按照系统要求正确安装；可以尝试使用包管理器安装，如Homebrew（macOS）或者Chocolatey（Windows）。
2.hexo-cli安装失败
解决方案：尝试使用管理员权限安装：
sudo npm install -g hexo-cli
检查npm配置，确保网络通畅，或使用淘宝镜像：
npm config set registry https://registry.npm.taobao.org
3.hexo generate`报错
解决方案：删除缓存和已生成的文件，然后重新生成：
hexo clean
hexo generate
检查文章文件的格式和命名是否正确，确保Markdown文件中没有语法错误。
